Basics

As mentioned on the homepage, a markup language is a computer language that uses tags to define elements within a document. HTML is one such language. To start your journey on website building with HTML, you need to understand what those tags are, and how to use them in order to construct a web page.

Tags

Tags are used to mark up the start of an HTML element and they are usually enclosed in angle brackets.1 For example: <article>. Most tags must be opened <article> and closed </article> in order to function. The information of such tags, such as a title, rests between the opening and closing tags. It is important to mention that when using multiple tags, the tags must be closed in the order in which they were opened.1 For example: <div><span>Basics</span></div>

Tags can also contain additional pieces of information, called attributes.1 These attributes are placed inside opening tags, like so: <a href="example.com" target="_blank">, where the link's location (href) and where to open the linked document to (target) are attributes of the <a> tag.

Construction

So you have learned what tags are, but that knowledge alone won't get you very far. What you now need to know, is how to use tags to build an actual web page. The simplest way to construct an HTML document goes as follows:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>Write a heading here.</h>
<p>Write a paragraph here.</p>
</body>
</html>

Some tags used in this example must be at the top of every HTML page you create. These tags are:

  1. <!DOCTYPE html>, a declaration that defines this is an HTML5 document
  2. <html>, which signals where you start writing in HTML
  3. <head>, which contains all the meta information about the page2 (data mostly meant for search engines and the like)
  4. <body>, here goes all the content for the page. It is a container for all the visible contents2

Inside the <head> tag, there are usually a few other tags that are also always included, such as:

  1. <title>, where the page name is inserted that appears at the top of the browser window or tab
  2. <meta>, where information about the document is stored, such as keywords, a description, et cetera

As aforementioned, in the <body> tag is where all the visible content of the web page goes. This includes such things as paragraphs, tables, images and more. The table below is an overview of elements that go into the <body> tag.

Element Purpose
<h1> to <h6> Headings, used to display titles or subtitles
<p> Paragraphs, used for writing text
<strong> Bold highlighted text, to show importance
<em> Emphasised text, to stress certain texts
<sub> Subscript text, like hellohello
<sup> Supscript text, like hellohello

This is a non-exhaustive overview, meaning there are more tags and more wonderful ways to format web pages. If you are interested in learning more, definitely do check out this website for a more elaborate explanation on the workings of HTML!
